
PASOS PARA INICIAR EL BOT:

1. Instala Node.js desde https://nodejs.org/
2. Abre una terminal en esta carpeta y ejecuta:
   npm install

3. Abre el archivo config.json y reemplaza "AQUI_VA_TU_TOKEN" con el token real de tu bot.

4. Ejecuta el bot con:
   node index.js

5. En Discord, escribe el comando:
   !work
